# Research Program Architecture


The present Trilogy belongs to a broader research program on latent cognition, bounded autonomy, and retrieval-native artificial intelligence. 
Within this program, Differentiable Latent Indexing in LLMs: Toward Native Retrieval as a Cognitive Function serves as the architectural foundation. 
That paper defines the core systems premise: retrieval should not remain an external augmentation layer, as in conventional RAG architectures, 
but may be modeled as a differentiable cognitive function embedded within the transformer’s internal representational dynamics.

The three hypothesis papers included as companion works extend this foundation into adjacent theoretical domains. 
They are not appendices to the present manuscript, nor preliminary drafts of the Trilogy. Rather, they function as formal satellite papers: 
each isolates one claim that supports, constrains, or expands the main theoretical sequence developed here.      

The JAIKE submission should be read as the architectural foundation for the Trilogy. 
While the present work develops the broader conceptual and cognitive implications of latent cognition, 
the DLI-LLM paper provides the systems-level mechanism through which retrieval, memory, 
and representational access become native architectural functions rather than external procedural services.

The three companion hypothesis papers serve a different function. They do not define the base architecture. 
Instead, they formulate testable or semi-formal theoretical extensions emerging from the same research program. 
Their role is to preserve conceptual precision around claims that would otherwise overload the main Trilogy narrative.
